# ledger-api — 2026-Jun-07

## 7f3a9c1 — Add /v2/transactions endpoint with cursor pagination

**What changed:** Added `GET /v2/transactions` returning cursor-paginated
entries; deprecated the unpaginated `/v1/transactions` (sunset in 90 days).

**Why:** `/v1` returned the full ledger in one payload and was timing out for
large accounts; cursor pagination caps response size.

**Files:** `src/routes/transactions.ts`, `src/pagination.ts`, `openapi.yaml`

**Flags / to-dos:** Contract change — `ledger-web` must migrate to `/v2`
before the `/v1` sunset. Cross-repo flag for the related `ledger-web` folder.
